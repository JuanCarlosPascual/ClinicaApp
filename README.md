# ClinicaApp

Sistema de administración de citas para un consultorio clínico desarrollado en Java.

## Instalación y configuración

### Requisitos previos
- Java JDK 26 o superior
- IntelliJ IDEA
- Git

### Pasos de instalación
1. Clonar el repositorio:
   git clone https://github.com/JuanCarlosPascual/ClinicaApp.git
2. Abrir el proyecto en IntelliJ IDEA.
3. Configurar el JDK en File > Project Structure > SDK.
4. Compilar el proyecto con Ctrl+F9.

## Uso del programa

1. Ejecutar el archivo JAR o correr Main.java desde IntelliJ.
2. Ingresar con usuario y contraseña de administrador.
3. Seleccionar una opción del menú:
   - 1. Registrar doctor
   - 2. Registrar paciente
   - 3. Crear cita
   - 4. Asignar cita a doctor y paciente
   - 5. Salir
